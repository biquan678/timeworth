# TimeWorth

Visual tools for money, time, and life decisions.

## Stack
- Next.js
- TypeScript
- Tailwind CSS
- Cloudflare Pages
- GitHub

## MVP
- Home
- 6 core calculator pages
- 5 guide pages
- About / Privacy / Terms

## Core routes
- `/life-in-weeks`
- `/time-value-calculator`
- `/save-x-per-day`
- `/savings-goal-calculator`
- `/fire-calculator`
- `/rent-vs-buy-calculator`

## Deployment
- GitHub for source control
- Cloudflare Pages for deployment
- Cloudflare DNS for domain

## Positioning
TimeWorth turns money, time, and life planning decisions into clear visual experiences.
